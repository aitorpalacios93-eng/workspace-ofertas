# State

Estado operativo persistente del workspace.

Archivos:

- `prospects.json`: oportunidades creadas desde la webapp o por importacion.
- `jobs.json`: ejecuciones del worker.
- `events.json`: timeline de eventos relevantes.
- `messages.json`: mensajes entrantes y borradores de respuesta.
- `settings.json`: preferencias operativas del dashboard.

Esta carpeta es el estado vivo del sistema, no su memoria estrategica.
