# 03 Propuesta y Cierre

## Regla maestra

La propuesta no explica todo lo que sabemos hacer. Explica por que esta oferta es la mejor siguiente jugada para este prospecto.

## Composicion obligatoria de la propuesta

1. Problema actual del prospecto.
2. Coste de no actuar.
3. Oportunidad concreta a corto plazo.
4. Oferta recomendada.
5. Alcance claro.
6. Resultado esperado razonable.
7. Inversion y modalidad.
8. Proximos pasos.

## Limites de composicion

- Una oferta core.
- Cero, uno o dos add-ons.
- Nada de catalogos infinitos.
- Nada de promesas imposibles ni ROI inventado.

## Reglas de ROI

El ROI se formula como escenario razonable, no como garantia.

- `ES_LOCAL`: llamadas, reservas, formularios, mejora de conversion local.
- `ES_B2B`: conversaciones cualificadas, velocidad comercial, autoridad, mejora de conversion web o de seguimiento.
- `US_HISPANIC`: booked calls, response speed, trust signals, conversion de lead a cita o a cliente.

## Battlecard

Cada oportunidad aprobada debe incluir una battlecard minima con:

1. Objeciones probables.
2. Respuesta marco.
3. Evidencia que falta.
4. Riesgo de descuento.
5. Siguiente pregunta a hacer.

## Creator-led content

`Creator-led content` es un acelerador opcional. Nunca se mete por defecto.

Solo aplica si:

- El negocio vende por confianza visual.
- Un rostro real mejora credibilidad.
- Hay una cara disponible y consistente para sostener la promesa.

## Follow-up

Toda propuesta completa debe generar:

- resumen ejecutivo corto
- deck spec
- ROI de reunion
- battlecard
- discovery questions
- 3 follow-ups

## Regla de honestidad comercial

Si el sistema no controla el trafico, el cierre o la operacion del cliente, no se promete resultado cerrado. Se promete mejora probable, ventana de recuperacion o escenario defendible.
